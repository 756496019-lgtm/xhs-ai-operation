一、第一次准备

整个准备只做一次，约 15 分钟。完成后全程靠对话使用。

1.1 装两个工具（让 Code Buddy 跑得起来）

打开 Code Buddy，把下面整段话复制粘贴进去发给它：

请帮我做以下准备工作：
1. 检查我电脑上是否装了 Python 和 ffmpeg
2. 如果没装，给我具体的安装指引（按我的操作系统）
3. 装好后告诉我下一步做什么

Code Buddy 会自己探测你的操作系统，给你贴合的指引。如果它让你做某件事（比如下载某个文件、点某个按钮），照做就行。

验证装好的标志：Code Buddy 跑完后会告诉你"两样都齐了"。

1.2 把项目交给 Code Buddy 一次

把工具文件夹（xhs_console_agent）放到一个好找的位置，比如桌面。

然后跟 Code Buddy 说：

我把游戏短视频生产工具放在了 [告诉它路径，比如 ~/Desktop/xhs_console_agent]。
请你：
1. 进入这个目录看看
2. 读 README.md 和 skills/ 目录下所有 SKILL.md 文档
3. 跑 pip install -r requirements.txt 装项目依赖
4. 装完告诉我你都准备好了什么能力

Code Buddy 会自己看完文档，告诉你它现在能帮你做什么（5 个能力：素材采集、文案、配音、剪辑、封面）。

不知道路径怎么填？把 xhs_console_agent 文件夹直接拖到 Code Buddy 对话框里——大多数Code Buddy 都支持，路径会自动出来。或者直接告诉它"我桌面上有一个叫 xhs_console_agent 的文件夹"。

1.3 中文字体（按需补）

只在生成出来的视频/封面里中文显示成方块（□□□）时才需要做。跟 Code Buddy 说：

我的视频/封面里中文显示成方块了，请帮我装一个支持中文的字体到项目的
assets/fonts/ 目录下，告诉我下载和移动的具体步骤。

Code Buddy 会引导你下载思源黑体之类的免费字体并放到正确位置。

二、工作原理

了解一下 Code Buddy 干活的方式，能让你更会跟它说话。

它会自己看 skill 文档

工具的每个能力（采集/文案/配音/剪辑/封面）都有一份 skills/XX/SKILL.md 说明书。你不需要看，Code Buddy 会自己读。你只需要描述你想要什么，Code Buddy 会判断要触发哪个 skill、跑什么命令。

它会自己跑命令、自己排错

  • 命令报错 → Code Buddy 自己看错误信息、自己调整、自己重试
  • 网络断了 → Code Buddy 会停下告诉你
  • 中间结果不理想 → 你直接说"重做一次，X 改成 Y"

你只在两种情况下需要动手

  1. 第一次准备（装 Python/ffmpeg + 把项目交给 Code Buddy 看一眼）
  2. 下载某些素材（比如对标视频需要登录小红书时配 Cookie——Code Buddy 会一步步引导）

三、最快用法

一句话出片

启动 Code Buddy，在工具文件夹里，直接说一句话：

"帮我出一条本周折扣视频，标题叫'本周5款骨折必买'。"

Code Buddy 会按 SKILL.md 自动走完五步：抓商店折扣 → 下预告片 → 生成文案 → 配音 → 剪视频 + 封面。

跑完它会告诉你视频和封面文件在哪，你直接拖到小红书发布器即可。

想加变化？再说一句

想要什么 跟 Code Buddy 这么说
换男声配音 "用阳光男声配音"（Code Buddy 会选合适的音色，比如云希）
不要配音 "不要配音，出无声+字幕版"
配音慢一点 "配音放慢 10%"
加 PS 平台 "加上 PS 折扣，PS Store 港服折扣页地址是 [粘贴你从浏览器复制的网址]"
让文案更地道 "先去对标几条小红书的同类视频，再写文案"（详见第六节）
用我做的封面底板 "用 assets/templates/我的风格.png 这个封面底板"（详见第七节）

可以一句话里说多个：

"用云健解说员男声配音，配音放慢 10%，标题写'魂游必玩TOP5'。"

一句话搞不定？继续聊就行

你：帮我出一条本周折扣视频。
Code Buddy：好的，请告诉我封面标题。
你：本周5款骨折必买
Code Buddy：[开始跑] ...
你：等等，加上 PS 折扣
Code Buddy：好的，请把 PS Store 折扣页的网址复制给我。
你：https://store.playstation.com/zh-hans-hk/category/...
Code Buddy：[继续跑]

把它当个会写代码的合作伙伴聊就行——不用记任何命令。

四、按题材的话术

不同题材给 Code Buddy 的话不一样。下面五个常见题材的样板，你直接照抄改一改：

4.1 折扣盘点

"帮我出一条本周折扣盘点视频，5 款，标题写'本周5款骨折必买'。"

如果想加 PS：

"帮我出本周折扣视频，包括 PS 平台，PS Store 港服折扣页地址是 [网址]，标题写'本周5款骨折必买'。"

4.2 新游评测

"帮我出本周新游评测视频，5 款，主题是'本周值得关注的5款新游：玩点速览'，标题写'本周5款新游'，用专业感的男声配音。"

4.3 品类盘点（魂系/JRPG/独立游戏…）

魂系样板：

"帮我出一条魂系游戏盘点视频，包含这5款：艾尔登法环、黑神话悟空、卧龙苍天陨落、匹诺曹的谎言、明末渊虚之羽。主题是'5款不容错过的魂系游戏：从硬核到入门'，标题'魂游5选'，用解说员风格的男声配音。"

JRPG 样板：

"帮我做一条 JRPG 推荐视频，列这几款：女神异闻录5、八方旅人2、轨迹系列... 主题是'2026年最值得入坑的JRPG'，标题'JRPG入坑指南'。"

4.4 怀旧回顾

"帮我出一条 PS3 神作回顾视频，包含：战神3、神秘海域2、美国末日、GTA5、刺客信条2。主题'PS3黄金时代的5款封神之作：现在还能玩到'，标题'PS3神作回顾'，用温暖感的男声配音。"

4.5 即将发售

"帮我出一条下月即将发售的游戏视频，5 款，标题写'下月必玩5款'。"

4.6 系列回顾（自己选定一个系列）

"帮我做一条塞尔达系列回顾视频，包含：旷野之息、王国之泪、天空之剑、风之杖、时之笛。主题'塞尔达系列里你最爱哪一作？'，标题'塞尔达全系列'。"

五、单独用某能力的话术

如果你只想用其中一个能力（比如只生成文案、只剪视频），下面是各能力的话术。

5.1 只采集素材

不剪视频，只想抓数据：

"帮我抓本周三大主机的折扣数据，10 款左右，存成数据文件给我。"

或者按游戏名查：

"帮我查这几个游戏的 Steam 数据：[游戏A、游戏B、游戏C...]，包括价格、封面、预告片，存到本地。"

5.2 只生成文案

已经有素材，只想生成文案：

"我已经在 [告诉它素材路径] 有素材数据了，帮我写一份关于'某主题'的文案，60-90 秒视频时长，存为 script.txt。"

或者用预设模板（不用 Code Buddy 写，省 token）：

"用折扣盘点的预设模板生成文案。" (Code Buddy 自己会找到本地素材库)

5.3 只生成配音

已经有 script.txt，只想生成配音：

"用 script.txt 生成配音，用云健解说员男声，存到 voice/ 目录。"

不知道挑哪个音色？看第九节，或者：

"我做的是怀旧主题，配音用什么音色合适？"

让 Code Buddy 帮你挑。

5.4 只剪视频

已经有 script.txt + pv_library/ + voice/，只想剪视频：

"用现有的 script.txt 和 voice/ 配音，把 pv_library/ 里的预告片剪成一条小红书视频。"

不要配音版：

"用现有的 script.txt 和 pv_library/ 剪一条无声+字幕版视频。"

5.5 只生成封面

最简单的用法（默认布局：红顶白字 + 拼图）：

"用本地素材库里的 6 个游戏生成一张封面，标题'本周5款骨折必买'，副标题'三主机折扣盘点'。"

游戏封面图自动从商店爬取——Code Buddy 已经在素材采集阶段拿到每个游戏的官方头图（Steam/Switch/PS 商店的标准封面），你不用自己找图。

底板长这样（中间留白等着贴游戏图）：

"用 steam_spring 这套底板做封面，6 款游戏，标题写 'steam春促桌面游戏推荐'，副标题 '找到6款你的赛博同桌'。"

做更多风格

5.6 已经有文案，从中间开始

如果你自己写好了文案 script.txt：

"我自己写好了文案 [告诉它路径]，从这里开始走流程，配音 + 剪视频 + 封面都做了，标题写'XXX'。"

5.7 做图文笔记（不是视频）

如果你不想做视频，只想做小红书图文笔记（一组 5-10 张图片），有三种玩法。

输出都是：1 张封面 + 几张内容卡片，每张 1080×1440 竖版 JPG，直接拖到小红书发布器按顺序选就行。

5.7.A 一条龙生成（和视频用同一份文案）

如果你已经走完视频流程，想顺便出一份图文版：

"用我刚才做的 script.txt 文案，再出一组图文笔记，每个游戏一张卡片，封面用第一个游戏的封面图当背景。"

工具会自动用每个游戏的官方封面作背景，文字（标题、价格、卖点）打在上面。

5.7.B 只给文案，让工具排版

适合你自己写好了内容（不一定关于游戏），直接交给工具排版成图文：

把文案存成一个 txt 文件，格式像这样：

本周5款骨折必买游戏

塞尔达传说 王国之泪
Switch平台年度神作，开放世界冒险天花板。原价519港币，现在折扣到349港币，相当于7折。

艾尔登法环
PC平台魂系巅峰之作，宫崎英高携手乔治马丁。Steam打折期间仅需149元，50%折扣。

黑神话悟空
国产3A标杆，西游神话动作冒险。

格式规则：
- 第一行是标题（封面用）
- 之后每段中间用空行分隔
- 每段第一行是小标题（卡片标题），后面几行是正文

然后跟 Code Buddy 说：

"把 content.txt 排成小红书图文笔记，副标题写'本周折扣盘点'。"

工具会自动：
- 第 1 张是封面（顶部标题色块 + 中间红色序号目录 + 底部副标题胶囊）
- 后面每段一张卡片（带页码徽章 1/N、标题、正文）

5.7.C 用 lightai 做底图，工具只打字

这是最自由的玩法——你用 lightai 出底图（任何风格都行），工具负责把文字精准打到底图上。

步骤 1：让 Code Buddy 帮你写给 lightai 的提示词，重点是底图本身要有设计感，同时预留文字排版区：

"我要做一组小红书图文笔记，主题是'______'。请帮我写给 lightai 的提示词，要 4 张 1080×1440 竖版底图（封面 + 3 张内容卡）。要求底图有完整的视觉设计：有页眉/页尾装饰条、有品牌色背景（风格参考：______），不要纯白/纯色空白底。底图中部预留文字排版区域（不需要真的留白，可以用浅色色块、半透明遮罩、或低对比度纹理来标示文字区），保证后期叠加文字时可读。"

上面 ______ 的地方按你的风格填写，比如：

  • "赛博朋克霓虹风，深色底+荧光色点缀"
  • "日系杂志感，奶油色调+几何色块分区"
  • "小红书爆款科技蓝，渐变蓝紫底+白色圆角卡片区"
  • "复古像素游戏风，深绿底+像素边框页眉页尾"

步骤 2：拿提示词去 lightai 生成图片，下载下来，放到项目某个目录，比如 my_bgs/。

步骤 3：准备好文案 content.txt（格式同 5.7.B），然后告诉 Code Buddy：

"我有 4 张 lightai 生成的底图在 my_bgs/ 目录下（按顺序 1.png 是封面、2-4.png 是内容卡片），文案在 content.txt，帮我把文字打到这些底图上，生成图文笔记。"

工具会自动加黑色半透明蒙版 + 白字描边，保证文字在任何底图上都可读。

怎么选模式

你的情况 用哪个
已经在做视频流水线，顺便出图文 5.7.A 一条龙
有写好的稿子，希望快速排版 5.7.B 只给文案

六、找对标参考（进阶）

这一节可选。基础流程不需要。
适合：模板看腻了想换花样、想知道同行怎么写、想要"借鉴结构但不抄文字"的高质量文案。

6.1 这是干啥的

让 Code Buddy 先去小红书 / B站 / YouTube 抓几条同行的对标视频字幕和文案，然后学他们的语气结构来写你的版本（不抄文字，只学结构）。

底层用 Agent Reach（开源工具，免费）。

6.2 装 Agent Reach（一次性）

跟 Code Buddy 说：

"帮我装 Agent Reach 工具，安装文档在 https://raw.githubusercontent.com/Panniantong/agent-reach/main/docs/install.md，装完跑 agent-reach doctor 看哪些平台已经能用。"

Code Buddy 会自动装、自动测、告诉你哪些平台开通了。

6.3 配小红书（一次性，可选）

小红书需要登录才能爬，跟 Code Buddy 说：

"帮我配置 Agent Reach 的小红书 Cookie。"

Code Buddy 会一步步引导（通常用 Cookie-Editor 浏览器插件导出 Cookie 文件）。Cookie 只存你电脑本地，不传给任何人。

6.4 用对话采集对标

直接告诉 Code Buddy 你想要什么对标：

"用 Agent Reach 在小红书搜'魂系游戏盘点'前 5 条最热的帖子，提取标题和正文，整理成 references.md 给我。"

或者抓某个具体视频：

"用 Agent Reach 把这条 B 站视频的字幕扒下来，存到 references.md：https://www.bilibili.com/video/BV..."

6.5 让对标参与你的视频

跟出片命令一起说：

"帮我做一条魂系盘点视频，先去小红书搜 5 条同类对标存到 references.md，然后让 Code Buddy 学他们的语气和结构（但不抄文字）写文案，用云健配音，标题'魂游5选'。"

Code Buddy 会按顺序：先采对标 → 再生成文案（带 references）→ 再走配音/剪辑/封面。

6.6 合规很重要

提醒 Code Buddy 注意（其实它一般会自己注意）：

  • ✅ 学开头钩子、节奏、语气、切入角度
  • ❌ 不要照搬整句、特殊词组——平台会判洗稿，限流
  • ❌ 不要把对标视频画面剪进自己视频——侵权
  • 🔒 Cookie 不要传给任何人

6.7 不用 Agent Reach 也行

最简单办法：你手动去小红书/B站找几条对标，手动把文案抄成 references.md，然后跟 Code Buddy 说：

"我准备好了对标参考在 references.md，帮我做一条 XX 视频，让文案学这份对标的语气结构（不抄文字）。"

效果一样。

七、做高质感封面（进阶）

这一节也可选。基础封面（红顶白字+拼图）够用。如果想要小红书爆款那种"质感封面"——背景有游戏氛围、标题底是质感色块/胶带——需要这一节。

7.1 思路

让 lightai 一次性做一张 1080×1440 的"封面底板"PNG，留出标题区、游戏区、副标题区的空白。然后代码每次出片时把游戏图和文字贴上去。

核心好处：底板做一次复用很久。每周出片只换游戏图和标题——不用每次烧 lightai 的额度，风格还稳定。

[lightai 做一次] → 底板.png + 布局.txt → [代码反复用] → 每次出新封面
                                              ↑
                                        当期游戏图 + 当期标题

7.2 步骤 1：让 Code Buddy 帮你出底板提示词

跟你的 Code Buddy 说：

"我想要一张暗调 Steam 风格的小红书游戏封面底板，1080x1440 竖版。请帮我写一份给 lightai 的提示词，要求：
- 顶部 320 像素留出标题色块（红色质感、带胶带感）
- 中间从 380 到 1300 像素是 6 个游戏封面位置（只画暗色背景，不画游戏）
- 底部 100 像素留副标题标签条
- 整体调性 [描述你想要的风格]
提示词必须强调：所有空白区域留空，不要写任何文字。"

它会给你一份贴合你需求的提示词。

7.3 步骤 2：用 lightai 出图

复制提示词到 lightai，生成 PNG。

把生成好的 PNG 保存到工具文件夹的 assets/templates/ 下，比如：
- assets/templates/dark_steam.png

文件名用英文，方便后续引用。

7.4 步骤 3：让 Code Buddy 帮你写布局文件

底板做好了，但代码不知道标题区、游戏区分别在底板的哪个像素位置。让 Code Buddy 帮你估：

"我做了一张封面底板 assets/templates/dark_steam.png，请帮我看这张图，估算标题区、游戏区、副标题区的坐标，写一份对应的 dark_steam.txt 布局文件。可以参考 assets/templates/example_layout.txt 的格式。"

Code Buddy 会读图、估坐标、生成对应的布局文件。

7.5 步骤 4：用底板出封面

"帮我做一条本周折扣视频，封面用 assets/templates/dark_steam.png 这个底板，标题'本周5款骨折必买'。"

或者只换封面：

"帮我重新做一下封面，用 dark_steam.png 底板，标题'本周5款骨折必买'。"

7.6 不满意就反馈

封面出来不对劲？直接告诉 Code Buddy：

"标题字位置太靠下了，往上挪一点。"
"游戏图覆盖了底板上的装饰元素，缩小一点。"
"副标题颜色太浅，加个黑色描边。"

Code Buddy 会改布局文件里的坐标重跑。

7.7 准备多套风格

可以做几套底板风格，按题材切换：

题材 推荐底板风格
折扣/新游 暗调 Steam 风（dark_steam）
怀旧回顾 复古风（retro）
魂系/硬核 暗黑奇幻（souls）
休闲/独立 可爱风（kawaii）

让 Code Buddy 帮你各做一套。每次跟 Code Buddy 说做哪个题材时，直接报底板名：

"做一条魂游盘点，封面用 souls.png 底板，标题'魂游必玩TOP5'。"

八、出错怎么办

Code Buddy 自己会处理大部分错误

本地 Code Buddy 会自动看错误信息、自己调整、自己重试。大多数情况你什么都不用做——它跑完了告诉你结果就行。

Code Buddy 卡住或问你怎么办时

它问什么你答什么。常见情况：

Code Buddy 问你 你回答
"网络好像断了，要重试吗？" "重试"
"Switch 接口变了，要我帮你修代码吗？" "好"
"这个游戏没找到预告片，跳过还是手动下？" "跳过"（除非你想手动补）
"封面字数太多被截断了，要缩短吗？" "标题改成 [更短的版本]"

Code Buddy 也搞不定时

跟它说：

"你刚才那步报错了，请详细说明：1) 错误是什么意思 2) 为什么会出 3) 至少给我两种修复思路。"

如果它已经在帮你修但反复失败，换个角度：

"我们换个方式做。这一步先跳过 / 用 [其他方法] 替代。"

重头再来

如果一切都乱了，跟 Code Buddy 说：

"把当前工作目录清掉，从头重新跑一遍 [描述你想要的视频]。"

附录：精品号运营心得

游戏自媒体想长期做下去，不能靠"灵感"，而要像游戏公司一样做工业化、可复制的内容运营。

核心认知

做不下去的主因不是赚不到钱，而是无法稳定输出内容。

把内容生死交给"灵感"，本质上是被动、不可持续的。精品号的解法是把创作流程拆解为可标准化的环节，其中人负责灵魂（选题/立意/脚本），工具负责体力（素材搜集/剪辑/动效）。

两大内容策略

策略 类比 做法 优势
重置老爆款 像游戏大厂重制旧作 把过去数据好但制作一般的视频，重新翻新画面、补完信息、重做节奏 选题已验证，降低扑街风险；翻新后可获得二次推荐
打爆精华合集 像出"年度版/全DLC合集" 把受过市场验证的中短视频做成大合集 提高收藏率和长尾流量；比从零做全新长视频省力数倍

AI 使用边界：越精品越要克制

精品号的原则：灵魂环节必须人做，体力环节才交给 AI。

环节 谁来做 理由
选题策划 人（必须） 选题决定内容天花板，AI 无法替代对受众和时机的判断
脚本/文案撰写 人（必须） 个人风格和观点是精品号的护城河，AI 写出来千篇一律
节奏/结构设计 人（必须） 开头钩子、转折节奏、结尾引导——这是创作者的手艺
画面风格定调 人（主导） 视觉调性是品牌记忆点，可用 AI 出参考但最终人决定
搜索/下载游戏素材 AI 辅助 批量搜 PV、截图、商店数据，纯体力活交给工具
粗剪/拼接时间线 AI 辅助 把素材按脚本顺序拼到一起，省去重复拖拽
PPT 类动画/字幕效果 AI 辅助 缩放、位移、渐显等模板化动效，工具批量生成更快
配音合成 AI 辅助 TTS 配音适合量产号；精品号建议真人配音或至少人工调音
封面制作 人 + AI 配合 AI 出底图/排版初稿，人微调文字和构图细节
数据复盘/选题库维护 人（必须） 哪条爆了、为什么爆、下一条怎么迭代——这是增长引擎

一句话总结：AI 是你的剪辑助手和素材搬运工，不是你的内容大脑。越想做精品，越要把"人的判断"从流程中摘出来保护好。
