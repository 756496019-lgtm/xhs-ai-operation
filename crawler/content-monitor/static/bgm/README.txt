内置曲库说明
=============

将以下 MP3 文件放入本目录，即可在视频生成页的"内置曲库"中使用：

  lofi_chill.mp3      — Lo-Fi 轻松（钢琴/爵士）
  epic_game.mp3       — 史诗游戏（管弦乐）
  anime_sparkle.mp3   — 二次元闪亮（轻快电音）
  tech_pulse.mp3      — 科技节拍（电子乐）
  cozy_cafe.mp3       — 咖啡厅氛围（吉他）

推荐来源（免版权音乐）：
  - https://pixabay.com/music/
  - https://freemusicarchive.org/
  - https://www.bensound.com/

文件不存在时，内置曲库选项仍会显示，但试听和下载会失败；
建议下载后重命名放入本目录即可生效，无需重启服务。
