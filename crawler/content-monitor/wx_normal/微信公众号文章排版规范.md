# 微信公众号文章排版规范

> 基于多次踩坑总结，下次直接按此规范产出，无需反复调试。

---

## 核心原则

微信公众号编辑器粘贴 HTML 时会**剥除大部分 CSS**，能保留的只有：

| 能保留 ✅ | 会被剥除 ❌ |
|---|---|
| `<td bgcolor="#颜色">` HTML 属性 | `background`（CSS 属性） |
| `style="background-color:#颜色"` （仅部分保留） | `background-image`、`linear-gradient` |
| `style="color:#颜色"` | `font-family` |
| `style="font-size"`、`font-weight` | `display:flex`、`position` |
| `style="padding"`、`margin` | `box-shadow`、`border-radius`（部分） |
| `<table>` 布局 | CSS 类名（class） |

**结论：凡是需要保留背景色的区域，必须同时写 `bgcolor` 属性 + `style="background-color:"`，缺一不可。**

---

## 标准结构模板

```html
<div id="article-body" style="max-width:677px;margin:0 auto;background-color:#ffffff;font-family:'PingFang SC','Hiragino Sans GB','Microsoft YaHei',sans-serif;line-height:1.8;word-break:break-word;">

  <!-- 所有内容放这里 -->

</div>
```

- 外层 `max-width:677px` 是公众号正文宽度
- `font-family` 写了也会被剥，但不影响发布后手机端显示（系统字体接管）

## 常用组件写法

### 1. 彩色 Banner 顶部（深色背景）

```html
<table cellpadding="0" cellspacing="0" width="100%" style="width:100%;margin:0;table-layout:fixed;">
  <tr>
    <td bgcolor="#E8820C" style="background-color:#E8820C;padding:10px 24px 6px;">
      <p style="margin:0;font-size:10px;color:#7A3800;letter-spacing:3px;font-weight:700;">副标题 / 栏目名</p>
    </td>
  </tr>
  <tr>
    <td bgcolor="#1a2840" style="background-color:#1a2840;padding:20px 24px 24px;">
      <p style="margin:0;font-size:24px;font-weight:900;color:#ffffff;line-height:1.35;">文章大标题</p>
      <p style="margin:10px 0 0;font-size:14px;color:#7a9bb0;">副标题说明</p>
    </td>
  </tr>
</table>
```

> ⚠️ **禁止用** `background:linear-gradient(...)` —— 微信不支持，整块会变白

### 6. 数据表格

```html
<table cellpadding="0" cellspacing="1" bgcolor="#e5e7eb" width="100%"
  style="width:100%;background-color:#e5e7eb;margin:0 0 18px;font-size:14px;table-layout:fixed;">
  <!-- 表头 -->
  <tr>
    <td bgcolor="#1a2840" style="background-color:#1a2840;padding:9px 14px;color:#F5B820;font-weight:700;">列1</td>
    <td bgcolor="#1a2840" style="background-color:#1a2840;padding:9px 14px;color:#F5B820;font-weight:700;text-align:center;">列2</td>
  </tr>
  <!-- 奇数行 -->
  <tr>
    <td bgcolor="#ffffff" style="background-color:#ffffff;padding:9px 14px;color:#374151;">内容</td>
    <td bgcolor="#ffffff" style="background-color:#ffffff;padding:9px 14px;color:#E8820C;font-weight:700;text-align:center;">值</td>
  </tr>
  <!-- 偶数行 -->
  <tr>
    <td bgcolor="#f9fafb" style="background-color:#f9fafb;padding:9px 14px;color:#374151;">内容</td>
    <td bgcolor="#f9fafb" style="background-color:#f9fafb;padding:9px 14px;color:#374151;text-align:center;">值</td>
  </tr>
</table>
```

> `cellspacing="1"` + 外层 `bgcolor="#e5e7eb"` = 产生 1px 分割线效果，无需 `border`

## 一键复制按钮（标准写法）

在 `<head>` 加样式，在 `<body>` 开头加按钮，在 `</body>` 前加脚本：

```html
<!-- head 内 -->
<style>
  #copy-btn {
    position:fixed;top:16px;right:16px;z-index:999;
    background:#E8820C;color:#fff;border:none;border-radius:8px;
    padding:10px 20px;font-size:14px;font-weight:700;cursor:pointer;
  }
  #copy-tip {
    position:fixed;top:58px;right:16px;z-index:999;
    background:#333;color:#4ADE80;border-radius:6px;
    padding:6px 14px;font-size:12px;display:none;
  }
</style>

<!-- body 开头 -->
<button id="copy-btn" onclick="copyArticle()">📋 复制到公众号</button>
<div id="copy-tip">✓ 已复制，去公众号编辑器 Ctrl+V</div>

<!-- body 结尾 -->
<script>
async function copyArticle() {
  const article = document.getElementById('article-body');
  const htmlStr = '<!DOCTYPE html><html><body>' + article.outerHTML + '</body></html>';
  const tip = document.getElementById('copy-tip');
  try {
    if (window.ClipboardItem && navigator.clipboard && navigator.clipboard.write) {
      const blob = new Blob([htmlStr], { type: 'text/html' });
      await navigator.clipboard.write([new ClipboardItem({ 'text/html': blob })]);
    } else {
      const range = document.createRange();
      range.selectNodeContents(article);
      const sel = window.getSelection();
      sel.removeAllRanges();
      sel.addRange(range);
      document.execCommand('copy');
      sel.removeAllRanges();
    }
    tip.style.display = 'block';
    setTimeout(() => tip.style.display = 'none', 3000);
  } catch(e) { alert('复制失败：' + e.message); }
}
</script>
```

## 使用流程

1. 新建 `wx_normal/文章名.html`，参照本文档和 `wechat_article_ai.html` 的结构写内容
2. 用浏览器打开 `http://localhost:5050/static/../wx_normal/文章名.html`（或直接双击）
3. 点右上角「📋 复制到公众号」
4. 在公众号编辑器 **Ctrl+V** 粘贴
5. 检查效果，封面图单独上传

> 注意：编辑器内字体会显示宋体，**发布后手机端自动变系统无衬线字体**，正常。
